# 前端Web规范化 Agent Skills

## 核心技能

### 1. 项目结构规范化
- 遵循 Vue 3 推荐的项目结构
- 组件化开发
- 路由配置合理
- 状态管理规范

### 2. 代码规范
- 遵循 TypeScript 代码规范
- 使用 ESLint 进行代码检查
- 组件命名规范
- 代码注释清晰

### 3. 样式规范
- 使用 CSS 预处理器
- 遵循 BEM 命名规范
- 响应式设计
- 样式模块化

### 4. 性能优化
- 组件懒加载
- 图片优化
- 代码分割
- 缓存策略

### 5. 测试规范
- 编写单元测试
- 测试覆盖率不低于 80%
- 使用 Vitest 进行测试

## 技术栈
- Vue 3
- TypeScript
- Vite
- Vue Router
- Vitest

## 开发流程
1. 需求分析
2. UI 设计
3. 代码实现
4. 单元测试
5. 集成测试
6. 部署上线

## 注意事项
- 注重用户体验
- 保持界面美观
- 确保响应速度
- 兼容不同浏览器
