# 后端Spring Boot规范化 Agent Skills

## 核心技能

### 1. 项目结构规范化
- 采用多模块结构，将业务逻辑拆分为多个微服务
- 遵循 Maven 标准目录结构
- 公共代码提取到 common 模块
- 工具类提取到 utils 模块

### 2. 配置管理
- 使用 application.yml 进行配置
- 支持多环境配置（dev、prod）
- 敏感信息通过环境变量注入

### 3. 代码规范
- 遵循 Java 代码规范
- 使用 Lombok 简化代码
- 采用 RESTful API 设计风格
- 统一异常处理

### 4. 数据库操作
- 使用 Spring Data JPA 进行 ORM 操作
- 数据库表结构设计合理
- 支持事务管理

### 5. 测试规范
- 编写单元测试
- 测试覆盖率不低于 80%
- 使用 JUnit 5 进行测试

## 技术栈
- Java 11+
- Spring Boot 2.5+
- Spring Data JPA
- MySQL
- Maven

## 开发流程
1. 需求分析
2. 技术方案设计
3. 代码实现
4. 单元测试
5. 集成测试
6. 部署上线

## 注意事项
- 遵循微服务设计原则
- 保持代码简洁明了
- 注重性能优化
- 确保系统稳定性
